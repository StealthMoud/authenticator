Cloud Sync Setup

1. Create Repo: Go to Github and make a Private repository (e.g., vault).
2. Find Settings: Top-right Profile > Settings > Developer settings (bottom left).
3. Get Token: Personal access tokens > Tokens (classic).
4. Configure: Generate new token (classic). Name it "Sync" and check the [x] repo checkbox.
5. Finish: Scroll down, click Generate token, copy it, and paste here.
